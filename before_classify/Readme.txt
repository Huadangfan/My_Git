#files_classify.csh是用来按照地震时间排序归类地震的SAC数据
#      执行命令为csh files_classify.csh [datafilename] [datafilename] [filelist_temp.txt]同时需要一个filelist_temp.txt文件
#      其中data为地震数据所放在的文件夹的文件名
#
#      filelist_temp.txt文件为你所下载的地震数据解压后的SAC文件名
#      filelist_temp.txt文件可以使用这个命令获取：1.先cd data
#                                                2.输入ls *SAC>../filelist_temp.txt
#
#执行成功之后即可！
